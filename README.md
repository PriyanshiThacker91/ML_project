# Cardiovascular Disease Prediction ML Project

## Objective
Predict cardiovascular disease using machine learning and deploy using Streamlit.

## Dataset
Kaggle cardiovascular dataset.

## Steps
1. Data loading and EDA
2. Feature engineering
3. Train test split
4. Model training
5. Model evaluation
6. Streamlit deployment

## Setup

pip install -r requirements.txt

Run training:
python train_model.py

Run app:
streamlit run app.py

## Output
User enters medical values and gets disease prediction.
